# Employee Performance Dashboard

## Project Overview
This is a full-stack Python web application for tracking employee performance metrics.  
It allows managers and HR teams to monitor employee productivity, tasks completed, performance scores, and last review dates.  

The dashboard provides an easy-to-use interface to **add employees**, **view metrics**, and **export employee data to JSON**.  
It uses **Flask** for the backend, **SQLite** for storage, and standard HTML/JS for the frontend.

---

## Features
- **Add Employees:** Enter employee name, team, tasks completed, and performance score.  
- **View Employee List:** Displays all employees in a table with key metrics.  
- **JSON Export:** Export all employee data to a JSON file for reporting or integration.
